<h1 align="center">thirdweb TypeScript SDK</h1>

A type-safe library to interact with any EVM-compatible blockchain in both Node.js and the browser.

Connect to user&rsquo;s wallets, interact with smart contracts, sign
messages, and utilize common standards such as tokens, NFTs, marketplaces; all with built-in RPC URLs, IPFS gateways, and more.

## How It Works

The TypeScript SDK provides a type-safe interface to interact with your smart contracts, wallets, and the blockchain.

Each [extension](/solidity/interfaces) that your smart contract implements unlock new functionality for you to use in the SDK, which
does the heavy lifting for you under the hood.
For example, if your smart contract implements the [`IMintableERC721`](/solidity/interfaces/erc721mintable) interface,
you unlock the ability to mint NFTs in a type-safe manner, using the [`contract.erc721.mint`](/typescript/sdk.erc721mintable) method.

Instantiate an instance of the [`ThirdwebSDK`](/typescript/sdk.thirdwebsdk) class to connect to your smart contract and start
interacting with it&rsquo;s extensions!
